# wawatiny

use tinypng.com to compress images

## Install

```
$ npm install -g wawatiny 
```

## How  to use ?

```
$ wawatiny start -i inputdirectory -o outputdirectory
```

or 

```
$ var wawatiny = require('wawatiny');
$ wawatiny({
$  input: 'inputdirectory',
$  output: 'outputdirectory'
$ })

```
